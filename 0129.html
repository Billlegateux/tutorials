Run RainLoop for Free Web Based E-mail

  1. Download RainLoop Download
  2. Extract RainLoop
  3. Copy RainLoop to a folder on a web server
  4. Open a browser and navigate to the location on the web server
    https://webdev.i12bretro.local/webmail/?admin
  5. Login as admin/12345
  6. Click Security in the side navigation
  7. Enter 12345 in the current password field and type a new secure password twice, click Update Password
  8. Click Domains in the side navigation > Add Domain
  9. Fill out the fields as shown below:
    IMAP Server: imap.i12bretro.local     port: 993
    SMTP Server: smtp.i12bretro.local    port: 465
    Secure: SSL/TLS
  10. Click Test, if the IMAP and SMTP headers both turn green the test was successful
  11. Click Add
  12. In a browser, navigate to https://i12bretro.local/webmail
  13. Enter an email address and password
  14. Click New Message
  15. In the To: field, type your locally hosted e-mail address
  16. Enter a test subject and body and click Send
  17. Enjoy your free browser based e-mail client
