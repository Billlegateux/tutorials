Univention Corporate Server Install Guide Using Virtualbox

  1. Open a web browser and download the Univention Corporate Server .ova Download
  2. Launch VirtualBox
  3. Select File > Import Appliance...
  4. Click the Browse icon > Navigate to the downloaded .ova file > Select UCS-Virtualbox-Image.ova
  5. Click Next and then Import
  6. Make sure the Uninvention Corporate Server VM is selected and click Start > Normal
  7. That's it
