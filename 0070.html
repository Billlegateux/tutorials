Running a Windows 98 VM in Virtualbox

  1. Download Microsoft Windows 98 Download
  2. Extract the Windows 98 files
  3. Launch Virtualbox
  4. Create a New VM by selecting Machine > New

    Name: Windows 98
    Machine Folder: C:\VMs
    Type: Microsoft Windows
    Version: Windows 98
    Memory Size: 512 MB
    Hard disk: Create a virtual hard disk now

  5. Click Create
  6. On the Create Virtual Hard Disk dialog

    Name the virtual disk image Windows 98.vdi
    File size: 1.00 GB
    Hard disk file type: VDI
    Storage on physical hard disk: Dynamically Allocated

  7. Click Create
  8. Select the VM and Click Settings
  9. Select Storage
  10. Click on the CD-ROM drive
  11. Select the disc dropdown to the right > Choose a virtual optical disc file...
  12. Browse to the extracted Windows 98 files and select Win98 .iso
  13. Click OK
  14. Make sure the Windows 98 VM is selected and click Start > Normal
  15. Press 2 to Boot from CD-ROM
  16. Press 1 to Start Windows 98 Setup from CD-ROM
  17. Press Enter to Continue with Setup 
  18. Press Enter to Configure unallocated disk space
  19. Select Yes to enable large disk support, press Enter
  20. Press Enter to continue
  21. Press 2 to Boot from CD-ROM
  22. Press 1 to Start Windows 98 Setup from CD-ROM
  23. Press Enter to Continue with Setup
  24. Click the Continue button to begin installing Windows
  25. Follow the prompts to give the machine a name and workgroup
  26. Wait for the files to copy and Windows 98 to complete initializing
  27. When the VM reboots, select 1 to Boot from the Hard Disk
  28. Enter a name and company on the User Information Dialog, click Next
  29. Select I accept the Agreement, click Next
  30. Enter a valid Windows 98 SE product key (H8J3X-9HQQG-DQXX7-6GFTJ-DF6MB), click Next
  31. Click Finish
  32. Set the date/time and select a timezone > Click Apply > Click OK
  33. Wait for Windows to Update Sytstem Settings
  34. At the login screen click OK
  35. Enjoy the boot up sound from the Gods
  36. Welcome back to Windows 98
  37. Revel in the nostalgia
