Running a Windows 2 VM in Virtualbox

  1. Download MS-DOS 6.22 Download
  2. Download Microsoft Windows 2 Download
  3. Extract the MS-DOS 6.22 and Windows 2 files
  4. Launch Virtualbox
  5. Create a New VM by selecting Machine > New

    Name: Microsoft Windows 2
    Machine Folder: C:\VMs
    Type: Microsoft Windows
    Version: Windows 3.1
    Memory Size: 32 MB
    Hard disk: Create a virtual hard disk now

  6. Click Create
  7. On the Create Virtual Hard Disk dialog

    Name the virtual disk image Microsoft Windows 2.vdi
    File size: 100 MB
    Hard disk file type: VDI
    Storage on physical hard disk: Dynamically Allocated

  8. Click Create
  9. Select the VM and Click Settings
  10. Select Storage
  11. Click on the floppy disk drive
  12. Select the disc dropdown to the right > Choose a virtual floppy disk file...
  13. Browse to the extracted MS-DOS 6.22 files and select Disk1.img
  14. Click OK
  15. Make sure the Windows 2 VM is selected and click Start > Normal
  16. Press Enter to start the installation and press Enter multiple times to accept the default values
  17. When prompted by the installer, go to the VM settings and swap out the MS-DOS disk images and press Enter to continue
  18. Once MS-DOS has been successfully installed, select Devices > Floppy Drives > Remove disk from virtual drive
  19. Press Enter in the VM to reboot
  20. Once MS-DOS has booted, select Devices > Floppy Drives > Choose disk image...
  21. Navigate to the extracted Windows 2 files and select D1_Setup.img
  22. Type the following commands
    A:
    setup
  23. The Windows 2 installation process will start
  24. Press Enter to continue
  25. Arrow down to select IBM, PC or 100% compatible
  26. Press Enter multiple times to accept the default values
  27. When prompted by the installer, go to the VM settings and swap out the Window 2 disk images and press Enter to continue
  28. Press F to finish setup
  29. Select Devices > Floppy Drives > Remove disk from virtual drive
  30. At the DOS prompt, type the following commands
    win
  31. Welcome back to Windows 2
  32. Launch Notepad.exe
  33. Select File > Open
  34. Navigate to the root of the C: drive and open autoexec.bat to edit it
  35. Add C:\Windows\win to the bottom of the file to automatically boot Windows 2 on system startup
  36. Select File > Save
  37. Select Machine > Reset > Reset
  38. Revel in the nostalgia
