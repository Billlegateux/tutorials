Installing Batocera on S905x Android TV Box (Sunvell T95x)

Things You Will Need

  • microSD card at least 4 GB

Downloads and Flashing to MicroSD

  1. Download Batocera Download
  2. Download Balena Etcher Download
  3. Run Balena Etcher
  4. Burn Batocera image to microSD card
  5. Safely remove microSD and re-insert
  6. Ignore/close the Windows dialogs to format the inserted microSD card
  7. Open the microSD card in Explorer and open the /boot directory
  8. Rename the .dtb file that corresponds to your Android TV box to DTB.IMG

    P200 = s905
    p201 = s905
    p212 = s905x
    p281 = s905w
    q201 = s912
    q200 = s912

Booting to Batocera for the First Time

  1. Plug the power adapter into the Android TV box
  2. If the device has never booted from external storage before the stock Android OS will load
  3. In Android, navigate to the application drawer > Run Update & Backup
  4. Select Local > Navigate to the microSD card > aml_autoscript.zip
  5. Select Update
  6. The system will reboot twice and should then begin booting Batocera from the microSD card
  7. You should see the Batocera splash screen
  8. Setup Your Controller
  9. Game
